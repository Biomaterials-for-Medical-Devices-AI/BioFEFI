biofefi.services.custom_loss module¶

biofefi.services.custom_loss.bayesian_regularization_loss(model, prior_mu: float = None, prior_sigma: float = None) → Tensor¶

Compute the Bayesian Regularization loss.

The loss is computed as the sum of squared differences between model parameters and their prior mean, scaled by the prior standard deviation.

Parameters:

  • prior_mu (float, optional) – The prior mean. Defaults

  • provided. (Defaults to self._opt.prior_sigma if not)

  • prior_sigma (float, optional) – The prior standard deviation.

  • provided.

Returns:

The computed regularization loss.

Return type:

torch.Tensor

Raises:

ValueError – If both prior_mu and prior_sigma are not provided.
